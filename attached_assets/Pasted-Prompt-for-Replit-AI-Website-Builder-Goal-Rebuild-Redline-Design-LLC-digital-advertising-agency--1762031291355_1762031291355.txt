Tech + Performance

Stack: Next.js 14 (App Router) + React + TypeScript + Tailwind CSS.

Animations: Framer Motion for section reveals (fade/slide/scale), micro-interactions on buttons/cards, parallax hero background, stat counters. Prefers subtle movement; respect reduced-motion.

Images: Use optimized next/image; prefer PNG for static brand assets.

Speed: Lighthouse 95+ on mobile; lazy-load media; code-split; prefetch above-the-fold fonts only.

Accessibility: WCAG 2.2 AA, keyboard-navigable, visible focus states, ARIA where needed.

Brand & UI

Palette: Deep charcoal/near-black background, high-contrast white text, accent red for CTAs, trims, and underline effects.

Typography: Strong display for headlines (e.g., Inter/Oswald), geometric sans for body (Inter).

Components: Rounded-2xl cards, soft drop-shadows, glassy headers on scroll, sticky top nav.
